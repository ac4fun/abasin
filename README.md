# 渔盆模型投资指导工具 (Fishing Basin Model)

这是一个基于招财大牛猫（股社区）“渔盆模型”逻辑开发的 A 股核心指数投资指导工具。

## 功能特点
- **趋势分析**：基于 20 日均线（MA20）生成 YES/NO 信号。
- **临界值计算**：计算明日信号翻转的临界价格。
- **估值分析**：获取核心指数的 PE(TTM) 及其在历史数据中的分位值。
- **自动报告**：生成 Markdown 格式的每日投资指导报告。

## 核心指数
- 上证 50 (000016)
- 沪深 300 (000300)
- 中证 500 (000905)
- 中证 1000 (000852)
- 创业板指 (399006)

## 安装与使用
1. 安装依赖：
   ```bash
   pip install akshare pandas
   ```
2. 运行工具：
   ```bash
   python main.py
   ```
